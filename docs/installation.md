# Installation Guide

This guide explains how to run RepoRanger locally.

---

## Requirements

- Modern web browser (Chrome, Firefox, Edge)
- Internet connection (for GitHub API)

---

## Steps

Clone the repository:

```bash
git clone https://github.com/KristiyanGeorgiev1996/RepoRanger.git
```

Enter the project folder:
```
cd RepoRanger
```

Open the main file:
```
index.html
```

You can open it by double-clicking the file or running:
```
open index.html
```

---

### Notes
RepoRanger works entirely client-side and does not require a backend server.
