# Usage Guide

This guide explains how to use RepoRanger.

---

## Step 1 – Enter GitHub Username

Type a GitHub username in the input field.

Example:
torvalds


---

## Step 2 – Fetch Repositories

Click the **Search** button.

RepoRanger will request data from the GitHub API.

---

## Step 3 – View Repository Data

You will see:

- ⭐ Stars
- 🍴 Forks
- ⏱ Last Push
- 📊 Charts

---

## Step 4 – Explore Repositories

Click on a repository to open it on GitHub.

---

## Notes

- Only **public repositories** can be displayed.
- Data refreshes when the page is reloaded.
